# Retail Sales Dashboard (Tableau)

This project analyzes retail sales data from a US-based superstore to uncover trends in sales, profitability, and customer segments. The analysis includes data cleaning, key business metrics, and multiple chart visualizations built with Tableau

Objectives
- Analyze sales and profit trends
- Identify top-performing categories/products
- Evaluate regional performance
- Understand return behavior

Tools Used
- Tableau
- Excel
- SQL

Dashboard Features
- KPI tracking
- Interactive filters
- Geographic analysis
- Product profitability analysis
- Return analysis

Key Insights
- Technology generated the highest overall profit margin.
- Furniture had the highest return-related losses.
- The West region contributed the highest sales revenue.
